Pachamama Hospedagem

Projeto desenvolvido em React + Vite para uma experiência moderna de hospedagem inspirada em plataformas como Booking e Airbnb.

A aplicação apresenta uma interface sofisticada, estrutura preparada para responsividade e focada em experiência do usuário para visualização de hospedagens, disponibilidade, agenda cultural e realização de reservas online.

Responsividade

A estrutura do projeto está sendo preparada para adaptação completa entre dispositivos mobile, tablet e desktop utilizando TailwindCSS.

Como executar o projeto:
Clonar repositório
git clone URL_DO_REPOSITORIO
Entrar na pasta
cd pachamama-react
Instalar dependências
npm install
Rodar aplicação
npm run dev

Dependências principais?
npm install react-calendar
npm install lucide-react
npm install ical.js
